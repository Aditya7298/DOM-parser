Parse headings, images, links from pages!

![Screenshot](/Resources/Screenshot-2.png)


## Setting up the extension - server
1. npm install
2. npm start

## Setting up the extension - server
1. go to `chrome://extensions`
2. turn on the `developer mode` toggle
3. `Load unpacked`
4. select `extension` folder
